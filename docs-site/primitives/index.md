# Primitives Docs

This portal covers all Affino primitives except DataGrid.

## What is inside

- **Core packages**: headless primitives (`surface-core`, `menu-core`, `tooltip-core`, `popover-core`, `dialog-core`, `selection-core`, `listbox-core`, `combobox-core`, `tabs-core`, `disclosure-core`, `grid-selection-core`, `virtualization-core`, `treeview-core`)
- **Infrastructure**: `overlay-kernel`, `overlay-host`, `aria-utils`, `focus-utils`
- **Systems**: Menu / Tooltip / Popover / Dialog / Tabs / Disclosure guides
- **Adapters**: Laravel and Vue integration docs
